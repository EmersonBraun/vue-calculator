* [GIT](https://git-scm.com/)

OR
* [YARN](https://yarnpkg.com/)

## Getting Started
### Clone the repository
```
git clone https://github.com/EmersonBraun/vue-calculator.git
```

### Install the dependencies
```
npm install
```
Or
```
yarn
```

### Compiles and hot-reloads for development
```
npm run serve
```
Or
```
yarn serve
```
